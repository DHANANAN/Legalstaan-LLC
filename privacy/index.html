Legalstaan Privacy Policy

Last updated: 2026-05-08 · App package: com.legalstaan.app

Legalstaan ("the App", "we", "our") is a legal-education Android application provided to law-exam aspirants. This policy explains what information we collect, why, and how it is handled.

1. Data We Collect

  • Email address & Google profile name — Firebase Authentication.
  • User profile, course progress, community posts — Cloud Firestore.
  • Push notification token — Firebase Cloud Messaging.
  • Anonymous usage analytics & crash diagnostics — Firebase Analytics.
  • Profile photo (optional) — only if you upload one.
  • AI assistant prompts — sent to Google Gemini API when you opt in via the AI+Web toggle.

2. How We Use Your Data

  • Authenticate users and protect accounts
  • Deliver lectures, materials, live classes
  • Send notifications about new content and live sessions
  • Diagnose crashes and improve performance
  • Operate the in-app community (posts, faculty DMs)
  • Generate AI study answers when opted in

3. Third-Party Services

  • Firebase Authentication, Firestore, Cloud Messaging, Analytics
  • Google Drive — streams pre-recorded lectures
  • Google Gemini API — powers the in-app AI study assistant

We do not sell user data and do not share data for advertising.

4. Children & Age

Targets users 16+. We do not knowingly collect data from children under 13.

5. Data Security

All traffic uses HTTPS/TLS. Firebase encrypts data in transit and at rest.

6. Account & Data Deletion

  • Online: Account deletion page
  • Email: krishnashelkeintern@gmail.com

We delete your data within 30 days of request.

7. Permissions

  • INTERNET, ACCESS_NETWORK_STATE — fetch content
  • CAMERA, RECORD_AUDIO — only for live class WebRTC
  • READ_MEDIA_IMAGES / READ_EXTERNAL_STORAGE — only to pick a profile photo
  • POST_NOTIFICATIONS — live class alerts

8. Changes to This Policy

We may update this policy. The "Last updated" date reflects the most recent revision.
